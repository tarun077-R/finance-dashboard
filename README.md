# 💰 Finance Dashboard

A responsive finance dashboard built using React and Tailwind CSS.

## 🚀 Features
- Add & delete transactions
- Category-based tracking
- Dynamic income & expense calculation
- Chart visualization (Chart.js)
- LocalStorage persistence
- Responsive UI

## 🛠 Tech Stack
- React
- Tailwind CSS
- recharts
